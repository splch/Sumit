Sumit Manual

Sumit will work regardless of the API; however, without it, the functionality will be limited.

1. Create Account

First, you need to go to Smmry to create an account.

2. Open Dashboard

Next, activate your account (by checking your email) and proceed to the.

3. Use the API key

Under the API box, note the API key. Simply input the API key into the top input box in Sumit's popup.

4. Enable Sumit

In the Allowlist textbox, add websites you wish to enable Sumit on. If you wish to enable Sumit on every site, type *.


You are now ready to take advantage of all Sumit has to offer! Hover over a link in an allowlisted website to preview its contents.
